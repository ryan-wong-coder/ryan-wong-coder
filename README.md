### 👨‍💻 About me

我专注于后端系统、分布式数据基础设施与开发者工具，喜欢把复杂的底层能力做成真正可部署、可验证、可观测、可维护的产品。

- ⚙️ 使用 **Java、Go、Rust、Python** 构建后端服务、系统组件与工程工具。
- 🗄️ 关注 **存储引擎、WAL、透明日志、存算分离与水平扩展**。
- 🔐 实践 **Merkle Tree、Ed25519、SHA-256、确定性编码与离线证明**。
- 🔌 能够完成从 **TCP / HTTP / gRPC、SDK、CLI 到桌面客户端** 的端到端实现。
- 📈 重视并发模型、性能测试、可观测性、备份恢复和工程质量。

`Go` · `TCP` · `JSON Lines` · `Event Loop` · `State Machine` · `Snapshot Storage`

`TypeScript` · `React` · `Electron` · `Node.js` · `IPC` · `Event-driven Architecture`
